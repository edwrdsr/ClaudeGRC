# ClaudeGRC

Portfolio project: Python CLI for automated GRC evidence collection from AWS and Claude-powered compliance analysis (SOC 2, NIST CSF, NIST AI RMF).

## Quickstart
1. Activate venv: `source .venv/bin/activate`
2. Run: `claudegrc collect --cloud aws --mock`
3. Analyze: `claudegrc analyze --ai-rmf`
4. Report: `claudegrc report`
